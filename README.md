# 初学Java案例

## 1. Git初始化

### 1.1 初始化本地仓库

```
git init
```

### 1.2 关联远程仓库

```
git remote add origin https://github.com/yourusername/javacode.git
```

### 1.3 添加并提交本地代码

```
git add .
git commit -m "Initial commit"
```

### 1.4 推送代码到远程仓库

```
git push -u origin master
```
